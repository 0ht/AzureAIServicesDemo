## 推奨アプローチ

可能な限り、スクリプトの代わりにJupyter Notebookを使用することを推奨します。Notebookの方が：

- ✅ ステップバイステップの実行が可能
- ✅ 途中での確認と修正が容易
- ✅ ドキュメンテーションが統合されている
- ✅ エラー時のデバッグが簡単
